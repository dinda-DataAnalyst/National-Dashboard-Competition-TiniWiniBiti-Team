# National-Statistic-Dashboard-Competition-2025
🚀 Power BI Dashboard: E-Commerce Analysis
Proud to present our project for the National Statistic Dashboard Competition 2025, held by the Statistics Department of Universitas Sebelas Maret.

📊 We built an interactive Power BI dashboard to analyze e-commerce transactions from multiple angles — including customer behavior, product trends, delivery performance, and customer satisfaction.

🔎 This project aims to support data-driven decision-making by uncovering insights that can boost sales performance, improve customer satisfaction, and expand market reach.

🗂️ The dashboard includes two main pages:
1. Transaction Analysis Page
2. Customer Satisfaction Analysis Page

✨ Dashboard Highlights:
• Interactive landing and home pages
• Clickable navigation buttons across report pages
• Dynamic filters by year and month
• User-friendly design with actionable insights
